# GakView
멀티플랫폼을 목표로 하고 있는 간단한 뷰어입니다.  
타겟 플랫폼은 Windows, Linux, Max OS, Android, IOS 입니다.  

# Install
### Dependency
* SDL2.0
* Boost

### Linux
g++을 사용할 경우  

    $ make g++

clang++을 사용할 경우  

    $ make clang++

# Usage
    $ gakview [file]
